## 1. 系统简介

T-Malls 是一个面向多角色的综合商城系统，支持买家浏览下单、商家入驻经营、
平台统一管理与客服售后协同。系统采用 Web 端访问，无需安装客户端，
打开浏览器即可使用。

**访问入口：**

- 买家端：`https://<域名>/`
- 商家后台：`https://<域名>/seller`
- 平台管理后台：`https://<域名>/admin`
- 客服工作台：`https://<域名>/service`

---

## 2. 角色总览

| 角色 | 主要职责 | 登录入口 |
| --- | --- | --- |
| 买家 | 浏览商品、下单、支付、评价、售后申请 | 买家端首页 |
| 商家 | 商品管理、订单处理、店铺装修、营销活动 | 商家后台 |
| 平台管理员 | 商家审核、商品审核、订单仲裁、平台运营配置 | 平台管理后台 |
| 客服 | 处理用户咨询、协助售后、纠纷跟进 | 客服工作台 |

系统角色关系如下图所示：

```
        ┌───────────┐        下单/评价        ┌───────────┐
        │   买家    │ ───────────────────▶  │   商家    │
        └─────┬─────┘                        └─────┬─────┘
              │                                    │
     咨询/售后 │                       审核/入驻     │
              ▼                                    ▼
        ┌───────────┐       监管/仲裁        ┌───────────┐
        │   客服    │ ◀──────────────────── │  管理员   │
        └───────────┘                        └───────────┘
```

## 3. 买家用户手册

### 3.1 注册与登录

**操作步骤：**

1. 打开买家端首页，点击右上角"注册"按钮；
2. 输入手机号并获取短信验证码，设置登录密码；
3. 注册完成后，使用手机号 + 密码登录；
4. 已有账号可直接点击"登录"。

**界面示意：**

![登录注册界面](./images/buyer-login.png)

> 截图说明：页面顶部为平台 Logo，中部为登录/注册切换选项卡，
> 下方为输入框与提交按钮。

---

### 3.2 浏览与搜索商品

**操作步骤：**

1. 首页按分类、热销、新品等模块展示商品；
2. 在顶部搜索框输入关键词，可按商品名、店铺名搜索；
3. 在结果列表可按价格、销量、评分排序；
4. 点击任意商品卡片进入商品详情页。

**界面示意：**

![首页与搜索结果](./images/buyer-home.png)

> 截图说明：顶部为搜索栏与分类导航，主体为商品瀑布流，
> 每个商品显示图片、名称、价格与销量。

---

### 3.3 查看商品详情

**操作步骤：**

1. 进入商品详情页，查看主图、规格、价格、库存、商家信息；
2. 下滑可查看商品详情图文、参数、用户评价；
3. 选择规格（如颜色、尺码），点击"加入购物车"或"立即购买"。

**界面示意：**

![商品详情页](./images/buyer-product.png)

> 截图说明：左侧为商品主图轮播，右侧为商品标题、价格、规格选择
> 与操作按钮，底部是评价与详情区。

---

### 3.4 购物车与下单

**操作步骤：**

1. 在购物车勾选需要结算的商品；
2. 点击"去结算"进入订单确认页；
3. 选择或新增收货地址；
4. 选择配送方式、使用优惠券（如有）；
5. 核对金额后点击"提交订单"。

**界面示意：**

![购物车](./images/buyer-cart.png)

![订单确认页](./images/buyer-checkout.png)

---

### 3.5 支付

**操作步骤：**

1. 提交订单后进入收银台；
2. 选择支付方式（微信 / 支付宝 / 银行卡）；
3. 按提示完成支付；
4. 支付成功后页面会自动跳转至订单详情。

**界面示意：**

![收银台](./images/buyer-pay.png)

> 温馨提示：如 15 分钟内未支付，订单将被自动取消，库存释放。

---

### 3.6 订单管理与物流查询

**操作步骤：**

1. 进入"我的订单"页面，按状态（全部 / 待支付 / 待发货 / 待收货 / 已完成）筛选；
2. 点击订单进入详情页，可查看物流信息；
3. 收到货后点击"确认收货"，订单进入"待评价"状态。

**界面示意：**

![我的订单](./images/buyer-orders.png)

---

### 3.7 评价与晒单

**操作步骤：**

1. 已完成的订单可进入"评价"页面；
2. 为商品打分（1–5 星），填写文字评价，可上传图片；
3. 提交后评价会显示在商品详情的评价区。

**界面示意：**

![评价提交](./images/buyer-review.png)

---

### 3.8 售后与退换货

**操作步骤：**

1. 进入订单详情，点击"申请售后"；
2. 选择售后类型：仅退款 / 退货退款 / 换货；
3. 填写原因、上传凭证图片；
4. 等待商家审核，审核通过后按提示寄回商品；
5. 如与商家协商不成，可申请客服介入。

**界面示意：**

![售后申请](./images/buyer-aftersale.png)

---

### 3.9 个人中心

个人中心集中管理账户相关信息，包括：

- 账户资料与头像修改
- 收货地址管理
- 优惠券 / 积分
- 收藏夹与浏览记录
- 账号安全设置（密码、绑定手机）

**界面示意：**

![个人中心](./images/buyer-profile.png)

## 4. 商家用户手册

### 4.1 入驻与登录

**操作步骤：**

1. 访问商家入驻页，提交营业执照、法人身份证等资质材料；
2. 等待平台审核（一般 1–3 个工作日）；
3. 审核通过后，系统将推送账号信息；
4. 使用商家账号登录商家后台。

**界面示意：**

![商家入驻](./images/seller-apply.png)

---

### 4.2 商家后台首页（工作台）

登录后进入工作台，主要信息区包括：

- **待办事项**：待发货订单、售后工单、待审核商品数量
- **经营概览**：今日销售额、访客数、转化率
- **数据趋势图**：近 7/30 天销售曲线

**界面示意：**

![商家工作台](./images/seller-dashboard.png)

---

### 4.3 商品管理

#### 4.3.1 新增商品

**操作步骤：**

1. 左侧菜单进入"商品管理 → 发布商品"；
2. 填写商品名称、类目、品牌；
3. 上传主图、详情图；
4. 设置规格（SKU）：颜色、尺码、价格、库存；
5. 填写商品详情富文本；
6. 提交后等待平台审核。

**界面示意：**

![发布商品](./images/seller-product-create.png)

#### 4.3.2 商品列表

- 可按上架 / 下架 / 审核中 / 驳回状态筛选；
- 支持批量上下架、修改价格、修改库存；
- 点击"编辑"可修改商品信息。

**界面示意：**

![商品列表](./images/seller-product-list.png)

---

### 4.4 订单处理

**操作步骤：**

1. 进入"订单管理"，查看不同状态的订单；
2. 对"待发货"订单，点击"发货"，填写物流公司与运单号；
3. 系统自动将订单状态流转为"已发货"，同步给买家；
4. 如买家未及时收货，系统将在 7 天后自动确认收货。

**界面示意：**

![订单管理](./images/seller-orders.png)

---

### 4.5 售后工单处理

**操作步骤：**

1. 进入"售后管理"查看买家提交的售后申请；
2. 审核申请原因与凭证：
   - 同意：按流程处理退款或收货；
   - 拒绝：填写拒绝原因，买家可再次申诉或申请客服介入；
3. 收到退回商品后，点击"确认收货并退款"。

**界面示意：**

![售后工单](./images/seller-aftersale.png)

---

### 4.6 店铺装修

**操作步骤：**

1. 进入"店铺 → 店铺装修"；
2. 选择模板，拖拽组件（轮播图、商品推荐、优惠券、公告等）；
3. 编辑各组件内容并预览；
4. 点击"发布"使装修生效。

**界面示意：**

![店铺装修](./images/seller-decorate.png)

---

### 4.7 营销活动

商家可在"营销中心"创建：

- 优惠券（满减 / 折扣 / 无门槛）
- 限时折扣
- 满减活动
- 参加平台大促报名

**界面示意：**

![营销中心](./images/seller-marketing.png)

---

### 4.8 数据统计

**操作步骤：**

1. 进入"数据中心"；
2. 查看销售概览、流量来源、商品排行、买家画像；
3. 可按日 / 周 / 月切换统计周期；
4. 支持导出 Excel。

**界面示意：**

![数据中心](./images/seller-analytics.png)

## 5. 平台管理员手册

### 5.1 登录后台

管理员账号由系统预置，无法自行注册。使用分配的账号登录管理后台。

**界面示意：**

![管理员登录](./images/admin-login.png)

---

### 5.2 商家审核

**操作步骤：**

1. 进入"商家管理 → 入驻审核"；
2. 查看待审核商家列表；
3. 点击查看资质材料；
4. 通过：自动开通商家账号并发送通知；
5. 驳回：填写驳回原因，商家可补充材料后再次提交。

**界面示意：**

![商家审核](./images/admin-seller-review.png)

---

### 5.3 商品审核

**操作步骤：**

1. 进入"商品管理 → 待审核商品"；
2. 检查商品图片、描述、类目是否合规；
3. 通过后商品即可上架；
4. 驳回需填写原因，商家收到通知后可修改重审。

**界面示意：**

![商品审核](./images/admin-product-review.png)

---

### 5.4 订单与纠纷仲裁

**操作步骤：**

1. 进入"订单管理 → 纠纷订单"；
2. 查看买卖双方提交的凭证与沟通记录；
3. 根据平台规则做出裁决：支持买家 / 支持商家 / 双方协商；
4. 系统自动执行裁决结果（退款、放款等）。

**界面示意：**

![纠纷仲裁](./images/admin-dispute.png)

---

### 5.5 用户管理

管理员可在"用户管理"中：

- 查看买家 / 商家账号信息；
- 封禁违规账号或解封；
- 重置密码；
- 查看登录日志。

**界面示意：**

![用户管理](./images/admin-users.png)

---

### 5.6 平台运营配置

在"系统设置"中管理：

- 首页 Banner、推荐位
- 商品类目
- 平台优惠券
- 公告与帮助中心内容
- 支付、物流、短信等参数

**界面示意：**

![系统设置](./images/admin-settings.png)

## 6. 客服用户手册

### 6.1 工作台概览

客服登录后进入工作台，可看到：

- 待接入会话数量
- 今日处理工单数
- 个人绩效指标

![客服工作台](./images/service-dashboard.png)

### 6.2 在线会话

2. 客服在对话窗口回复消息，可发送文字、图片、商品卡片；

4. 问题解决后关闭工单并填写处理结果。

## 7. 常见问题 FAQ

**买家常见问题**

- **Q：下单后多久必须付款？**
  A：提交订单后 15 分钟内需完成支付，否则订单自动取消。
- **Q：怎么查看物流？**
  A："我的订单 → 订单详情 → 查看物流"。
- **Q：商品不满意怎么退货？**
  A：在订单详情点击"申请售后"，选择退货退款即可。

**商家常见问题**

- **Q：商品审核需要多久？**
  A：一般 1 个工作日内完成。
- **Q：订单 48 小时未发货会怎样？**
  A：系统会扣除保证金并向买家发放补偿券，请及时发货。
- **Q：如何参加平台大促？**
  A："营销中心 → 平台活动"报名，审核通过即可参加。

**管理员常见问题**

- **Q：如何批量处理商家审核？**
  A：在审核列表勾选多个商家后点击"批量通过 / 批量驳回"。
- **Q：裁决后能撤销吗？**
  A：裁决结果 24 小时内可由主管账号撤销并重新处理。

**客服常见问题**

- **Q：会话被买家挂起怎么办？**
  A：5 分钟无响应系统会自动结束会话并计入超时。
- **Q：如何升级纠纷？**
  A：在工单中点击"升级仲裁"，填写理由后提交管理员。
